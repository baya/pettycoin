About Pettycoin

Pettycoin is an experimental project to create a off-bitcoin network for small bitcoin transactions. (Click for more...)

Gateways pass bitcoins between the bitcoin and pettycoin networks. Pettycoin uses a cut-down bitcoin transactions, but the protocol is changed so that nodes don't need to know the entire blockchain to report problems.

The current code runs only on the test network, and is not ready for real usage! For more answers, see the FAQ.

Download

Pettycoin is Free and Open Source, and the source code is available on GitHub!

Development Information

Pettycoin is under active development, as you can read on the development blog. One day there will also be a mailing list to ask questions, so people don't have to use the bugtracker.
